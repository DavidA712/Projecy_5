Overview

This project analyzes travel behavior and vehicle ownership patterns in the Midwest using data from the National Household Travel Survey (NHTS) 2022. The analysis focuses on how household income influences:

Alternative fuel vehicle adoption
Vehicle ownership composition
Weekly travel behavior patterns


Research Questions
  1) How does the adoption of alternative fuel vehicles vary across household income groups in the Midwest?
  2) How does travel behavior vary across household income groups in the Midwest?

Key Findings
Conventional fuel vehicles remain dominant across all income groups.
Higher-income households show relatively greater adoption of alternative fuel vehicles.
Middle-income households demonstrate particularly high ownership of pickup trucks.
Higher-income households exhibit increased weekend travel behavior.
Travel behavior patterns vary significantly across socioeconomic groups.

Reproducibility

All analysis was completed in a Jupyter Notebook environment using Python. The workflow is reproducible provided the dataset and required libraries are available.
